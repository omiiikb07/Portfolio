## Things to Add for Portfolio Page
- Projects: short descriptions and links
- Case studies: problem, approach, results
- About: short bio and photo
- Skills: tech stack and tools
- Resume/CV: downloadable PDF
- Contact: email, LinkedIn, GitHub
- Blog/Notes: optional writing
- Testimonials: client or colleague quotes
- Gallery: screenshots, demos, or videos
- Hire me: clear call-to-action / contact info
